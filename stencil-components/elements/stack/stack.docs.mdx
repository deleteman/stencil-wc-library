## Overview
Basic Stencil component created with bit. This is a container component that stacks whatever is inside it either vertically or horizontally.

#### Example

Usage example:

```js
import '@teambit/web-components.stencil.stack/stack.tsx';

<st-stack direction="vertical">
<p>first element</p>
<p>second one</p>
</st-stack>
```
